

# Ruby 
Ruby is a web-based PCB Repair & Production Tracking System designed to streamline and digitize the PCB refurbishment workflow. The platform enables organizations to track PCB lots throughout multiple processing stages, monitor pass/fail statistics, identify bottlenecks, and measure engineer performance using data-driven insights.

Features
Lot-wise PCB tracking from receipt to final quality check
Real-time monitoring of PCB movement across 13 processing stages
Stage-wise pass, fail, and pending quantity tracking
Failure analysis to identify high-defect stages
Engineer assignment and productivity tracking
Performance leaderboards and monthly analytics
Company-wise repair and success rate reports
Interactive dashboards with visual charts and insights
Role-based access for Admins, Supervisors, Engineers, and Management
Automated reporting and operational analytics
Tech Stack
Frontend: React.js, Tailwind CSS
Backend: Node.js, Express.js
Database: PostgreSQL
Authentication: JWT
Visualization: Chart.js / Recharts
Objective

Ruby aims to improve operational visibility, reduce manual tracking efforts, identify process bottlenecks, and provide actionable insights for both production management and employee performance evaluation in PCB repair and manufacturing environments.
