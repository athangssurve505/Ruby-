

# Ruby 
Ruby is a web-based PCB Repair & Production Tracking System designed to streamline and digitize the PCB refurbishment workflow. The platform enables organizations to track PCB lots throughout multiple processing stages, monitor pass/fail statistics, identify bottlenecks, and measure engineer performance using data-driven insights.

## ✨ Features

- 📦 Lot-wise PCB tracking from receipt to final quality check
- 🔄 Real-time monitoring of PCB movement across processing stages
- 📊 Stage-wise pass, fail, and pending quantity tracking
- ⚠️ Failure analysis to identify high-defect stages and process bottlenecks
- 👨‍🔧 Engineer assignment and productivity tracking
- 🏆 Performance leaderboards and monthly analytics
- 🏭 Company-wise repair statistics and success rate reports
- 📈 Interactive dashboards with charts and operational insights
- 🔐 Role-based access control (Admin, Supervisor, Engineer, Management)
- 📑 Automated reporting and business analytics
- ⚙️ Dynamic workflow configuration for company-specific repair processes

## 🎯 Project Objective

Ruby is a PCB Workflow Management System that digitizes and streamlines PCB repair operations. The platform enables organizations to track PCB lots across customizable workflows, monitor stage-wise performance, identify failure trends, and evaluate engineer productivity through real-time analytics and reporting.

By providing complete visibility into the repair lifecycle, Ruby helps organizations improve operational efficiency, reduce process bottlenecks, and make data-driven decisions.
